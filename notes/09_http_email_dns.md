# HTTP, Email, and DNS

## HTTP (Hypertext Transfer Protocol)
- Foundation of web communication.
- Uses TCP.
- Methods: GET, POST, PUT, DELETE (RESTful APIs).
- Status codes: 1xx (Info), 2xx (Success), 3xx (Redirect), 4xx (Client Error), 5xx (Server Error).

---

## Cookies
- Small text files stored by websites on your device.
- Used for login, preferences, tracking.
- Third-party cookies: Set by domains other than the one visited (ads, analytics).

---

## Email Protocols
- **SMTP:** Sending emails (client → server → recipient server).
- **POP3:** Retrieves emails, usually downloads and deletes from server.
- **IMAP:** Retrieves and manages emails, keeps them on the server for access from multiple devices.

---

## DNS (Domain Name System)
- Translates domain names to IP addresses.
- Root DNS servers direct queries to TLD servers.
