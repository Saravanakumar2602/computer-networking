# Computer Networks Notes

This collection of notes covers the basics of computer networking, organized into separate topics for easy reference. Each `.md` file focuses on a specific area:

- **01_protocols.md**: Overview of protocols (TCP, UDP, HTTP, ports, sockets, ephemeral ports).
- **02_ip_addresses.md**: IP addresses (local/private, global/public, NAT, ISP, Mbps/Gbps/Kbps).
- **03_network_types_topologies.md**: Types of networks (LAN, MAN, WAN), network topologies (bus, ring, star, tree, mesh), and transmission media.
- **04_osi_tcpip_models.md**: OSI and TCP/IP models, layer functions, client-server vs peer-to-peer.
- **05_network_devices.md**: Network devices (repeater, hub, bridge, switch, gateway, brouter).
- **06_transport_layer.md**: Transport layer concepts (TCP, UDP, multiplexing/demultiplexing, port numbers).
- **07_network_layer.md**: Network layer (routing, IPv4/IPv6, subnetting, packets, TTL, NAT, DHCP, firewalls).
- **08_critical_protocols.md**: Critical protocols and concepts (ARP, ICMP, OSI vs TCP/IP, IPv6 transition, CIDR).
- **09_http_email_dns.md**: HTTP, email protocols (SMTP, POP3, IMAP), cookies, DNS.

Refer to each file for detailed explanations and examples on each topic.
