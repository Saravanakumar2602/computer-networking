# Computer Networks Notes

Welcome to the complete course notes for computer networking, organized for easy navigation and deep learning. Each markdown file in the `notes/` folder is a detailed chapter, with explanations, diagrams, real-world examples, and practical relevance.

## 📚 Course Structure


Each file is a standalone chapter. You can read them in order or jump to any topic as needed:

- [01_protocols.md](notes/01_protocols.md): Protocols (TCP, UDP, HTTP, FTP, SMTP, POP3, IMAP, DHCP, DNS, SSH, Telnet, VNC, SNMP, HTTPS, NTP, sockets, ports, ephemeral ports, protocol stack)
- [02_ip_addresses.md](notes/02_ip_addresses.md): IP addresses (IPv4, IPv6, public/private, NAT, ISP, subnetting, reserved addresses, data rates)
- [03_network_types_topologies.md](notes/03_network_types_topologies.md): Network types (LAN, MAN, WAN), topologies (bus, ring, star, tree, mesh), physical/wireless media
- [04_osi_tcpip_models.md](notes/04_osi_tcpip_models.md): OSI and TCP/IP models, layer functions, client-server vs peer-to-peer, real-world examples
- [05_network_devices.md](notes/05_network_devices.md): Network devices (repeater, hub, bridge, switch, router, gateway, brouter, modem, access point, firewall)
- [06_transport_layer.md](notes/06_transport_layer.md): Transport layer (TCP, UDP, multiplexing/demultiplexing, error detection, flow/congestion control)
- [07_network_layer.md](notes/07_network_layer.md): Network layer (routing, IPv4/IPv6, subnetting, CIDR, NAT, DHCP, ICMP, ARP, firewalls, middleboxes)
- [08_critical_protocols.md](notes/08_critical_protocols.md): Critical protocols and concepts (ARP, ICMP, OSI vs TCP/IP, IPv6 transition, CIDR, DNS, NAT, DHCP, VLAN)
- [09_http_email_dns.md](notes/09_http_email_dns.md): Application layer protocols (HTTP, cookies, email protocols, DNS)
